# 轻卡 (LightCalorie) - UI设计稿

## 一、配色方案

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | `#FF6B35` | 橙色，活力/燃脂感 |
| 辅助色 | `#4ECDC4` | 青绿，健康/清新 |
| 强调色 | `#FFE66D` | 黄色，打卡成就 |
| 背景色 | `#FAFAFA` | 浅灰白 |
| 深色文字 | `#2D3436` | 主要文字 |
| 次要文字 | `#636E72` | 说明文字 |
| 成功色 | `#00B894` | 完成/达标 |
| 警告色 | `#FDCB6E` | 接近目标 |
| 危险色 | `#E17055` | 超标/未完成 |

## 五、组件清单

| 组件 | 说明 | 状态 |
|------|------|------|
| BottomNav | 底部导航栏 | default / active |
| CalorieRing | 热量环形进度条 | normal / warning / over |
| WeightTrendChart | 体重趋势折线图 | 有数据 / 空状态 |
| CheckInCard | 打卡任务卡片 | done / undone / missed |
| CalendarStrip | 日历条 | normal / checkin / today / missed |
| FoodItem | 食物条目 | 可编辑 / 可删除 |
| QuickAction | 快捷打卡按钮 | default / done |
| ProgressBar | 进度条 | 0-100% |
| TagBadge | 标签徽章 | 多种颜色 |
| ActionButton | 主按钮 | normal / loading / disabled |
| InputField | 输入框 | default / focus / error |
| UserCard | 用户信息卡片 | 带头像/无头像 |
